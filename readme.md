# Blog-Posts

Full-stack app written in React and Nest JS

## Set-up Nest Js API server

change directory to the api:

```bash
cd packages/api
```

run migrations

```bash
yarn migration:run
```

## Run client and API

```bash
yarn start
```

## Run only client

```bash
yarn start:client
```

## Run only API

```bash
yarn start:api
```
